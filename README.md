# Dự án Nguyên lý Hệ điều hành: TPCShell (Windows API)

## 👥 Thành viên nhóm
1. **Tuấn (Leader)** - Quản lý chung, REPL, Parser, Biến môi trường (`path`).
2. **Phương** - Core Process Execution (`CreateProcess`, Foreground/Background, Batch file).
3. **Chính** - Process Controller (`Kill`, `Stop`, `List`) & Signal Handler (`Ctrl+C`).

## 🛠️ Môi trường phát triển
- Ngôn ngữ: C/C++
- Công cụ: Visual Studio (Windows SDK)
